# Como usar:
- Baixe o **ngrok-helper.exe**, coloque em uma pasta.
- Na mesma pasta, baixe o [ngrok](https://ngrok.com/) e abra o **ngrok-helper.exe**.
- O resto é com você, tudo esta disponivel no [ngrok](https://ngrok.com/).
